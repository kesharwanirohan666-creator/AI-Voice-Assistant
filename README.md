# AI Voice Assistant
Jarvis AI Voice Assistant is a Python-based desktop assistant that performs tasks through voice commands. The project uses speech recognition and text-to-speech technology to interact with the user in real time.  The assistant can recognize voice commands and perform actions such as opening websites, telling the current time, and responding with voice output. It demonstrates the integration of voice input, command processing, and automated task execution.  Technologies used in this project include Python, SpeechRecognition, pyttsx3, sounddevice, NumPy, and webbrowser. The system records audio from the microphone, converts speech into text, processes commands, and responds using voice output.  This project was built as part of learning Artificial Intelligence concepts and building real-world Python automation tools.
run command in terminal:- python jarvis_gui.py
